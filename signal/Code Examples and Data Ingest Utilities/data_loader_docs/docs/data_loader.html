inspect_signal(index, modulation, snr, cdata, time, number_of_samples_in_instance, sample_rate, start_freq, stop_freq, interactive=None)¶

inspects a specific instance within the collection. Plots FFT and I,Q. Prints to console raw I Q amplitudes

Parameters:
  • index (int) – index of instance in collection
  • modulation (str) – the modulation schema of the instnace
  • snr (int) – the SNR of the instance
  • cdata (numpy.array dtype='complex64') – the complex samples for the instance
  • time (list of type float) – time series values
  • number_of_samples_in_instance (int) – number of samples in the instance
  • sample_rate (float) – sampling rate of the collect
  • start_freq (int) – start frequency in Hz of the collect
  • stop_freq (int) – stop frequency in Hz of the collect
  • interactive (bool) – optional value for display of each plot (if True) requires user to close plot before next is displayed)

N/A
